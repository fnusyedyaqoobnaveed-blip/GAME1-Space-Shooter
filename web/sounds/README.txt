Put your audio files HERE with these EXACT names:

  music.mp3          (background music - required for music to play)
  shoot.mp3          (when you fire)
  hit.mp3            (asteroid hits ship)
  game_over.mp3      (game over)
  monster_ship_crash.mp3   (monster hits ship)

Optional: music.ogg (fallback for music)

The game loads from the "sounds" folder next to index.html.
If you opened the game via http://localhost:8000, make sure you ran
the server from the "web" folder so that this "sounds" folder is
inside the same folder as index.html.
