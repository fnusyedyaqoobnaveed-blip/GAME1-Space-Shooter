# Spaceship Journey — Browser version

No Unity. Edit in Cursor, play in the browser.

## How to run

1. Open the **web** folder in Finder (or File Explorer).
2. **Double‑click `index.html`** to open it in your browser.

Or from Cursor: right‑click `index.html` → **Reveal in Finder** → double‑click `index.html`.

## How to test changes

1. Edit `game.js` or `index.html` in Cursor.
2. Save.
3. In the browser, **refresh the page** (F5 or Cmd+R).

That’s it. No build step, no Unity.

## Adding your own images (ship, monsters, target, etc.)

See **ASSETS.md** for how to add image files. Put PNGs in an **images** folder with names like `ship.png`, `monster.png`, `target.png`; the game will use them instead of the default shapes.

## Controls (also shown on screen)

When you open the game, **controls are shown on the left** and **what to do on the right**:

- **W A S D** or **arrow keys** — Move (up, left, down, right)
- **Space** — Shoot
- **Q** or **E** — Dodge (barrel roll; you can’t be hit while spinning)
- **Shift** — Boost (move faster for a short time)
- **P** or **Esc** — Pause / Resume

**Goal:** Fly to the **green circle**. Dodge **gray** (asteroids), shoot **red** (monsters), collect **yellow** (planets). The orange dot is your fairy.
