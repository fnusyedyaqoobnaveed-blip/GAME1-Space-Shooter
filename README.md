# Unity version (optional)

A 2D/3D spaceship game where you travel toward distant targets (in light years), dodge asteroids, fight monsters, get help from a fairy, and collect planets as coins. Difficulty scales as you progress.

---

## 🎮 Game Overview

| Element | Description |
|--------|-------------|
| **Goal** | Reach the target (N light years away) |
| **Movement** | Up, down, sideways — dodge and perform tricks |
| **Obstacles** | Asteroids (dodge) |
| **Enemies** | Monsters (hit/kill) |
| **Helper** | Fairy guides/helps toward target |
| **Collectibles** | Planets = coins |
| **Progression** | Difficulty ↑, target distance ↑, player speed ↑ |
| **Shooting** | Space/click to fire; damage & triple shot scale with level |
| **Tricks** | Barrel roll (Q/E) = invincibility; Shift = speed boost; P/Esc = pause/resume (web) |
| **Combo** | Chain monster kills for bonus planets (every 3 kills) |

## ⚡ Quick Start in Unity

1. Open **Unity Hub** → **New Project** (Unity 2021.3 LTS or newer).
2. Choose **2D** or **3D** template (2D is easier to start).
3. Create folders: `Scripts`, `Prefabs`, `Scenes`, `Sprites` (or `Models` for 3D).
4. Copy the scripts from **Scripts/** into your project’s `Assets/Scripts` folder.
5. Follow **GETTING_STARTED.md** to build your first playable scene.

Start with **GETTING_STARTED.md**, then use **Scripts/** and **GAME_DESIGN.md** as you build.
